## Dependencies
- `bash`
- `ncurses-utils` (Termux or OpenSUSE)

## Installation
```zsh
git clone https://github.com/creepy0964/housefetch
cd housefetch
chmod +x install
sudo ./install
#if you using termux run ./install -t
```
## Configuration
Edit your ``$HOME/.config/housefetch/config ``
